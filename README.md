# AI-Powered Glassdoor Salary Prediction System

An end-to-end Data Science and AI/ML web application that predicts salaries based on technical skills using Machine Learning and the Glassdoor Jobs Dataset.

---

## 🚀 Live Demo

Add your Render deployment link here:

https://glassdoor-ai-salary-predictor.onrender.com/

---

## 📌 Project Overview

This project analyzes Glassdoor job data to understand salary trends and predict salaries based on technical skills such as Python, SQL, AWS, Machine Learning, Tableau, and more.

The project combines:

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Machine Learning
- Flask Web Development
- Professional Frontend UI
- Deployment on Render

---

# 🎯 Business Problem

Companies and job seekers often struggle to understand salary trends across industries, skills, company sizes, and locations.

This project helps:

- Predict salaries using AI/ML
- Analyze salary trends
- Understand skill impact on compensation
- Visualize business insights from Glassdoor data

# 💻 Installation & Setup

## 1. Clone Repository

```bash
git clone https://github.com/yourusername/glassdoor-ai-salary-predictor.git
```

---

## 2. Navigate to Project Folder

```bash
cd glassdoor-ai-salary-predictor
```

---

## 3. Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

---

## 4. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# ▶️ Run the Project

## Train ML Model

```bash
python train_model.py
```

---

## Run EDA

```bash
python eda.py
```

---

## Start Flask Server

```bash
python app.py
```
